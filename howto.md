# Инструкция: Мониторинг процессоров NiFi по группам в Grafana

---

## Общая схема

Мониторинг строится на отправке в логи карты **групп → процессоров** через механизм **Controller Settings → Reporting Task**. Данные обновляются каждые **30 минут**.

---

## 1. Настройка Reporting Task в NiFi

1. Откройте **Controller Settings** (меню «гамбургер» → Controller Settings).
2. Перейдите на вкладку **Reporting Tasks**.
3. В настройках задачи укажите **список целевых групп**, по которым нужно выводить процессоры.
4. Задача автоматически раз в **30 минут** отправляет в логи актуальный список процессоров для каждой указанной группы.

---

## 2. Ограничение: не более 100 процессоров на группу

> **Важно:** На одну группу выводится **максимум 100 процессоров**. Если процессоров больше — список **обрезается**.

### Что делать, если процессоров > 100:

- **Вариант А** — структурировать группу: вынести часть процессоров во **вложенные подгруппы**, чтобы в каждой было ≤ 100.
- **Вариант Б** — разбить группу на **две отдельные группы**, каждая ≤ 100 процессоров.

---

## 3. Получение списка процессоров в Grafana

1. Откройте Grafana по **ссылке** (ссылка_1).
2. В селекторе временного диапазона выберите **последние 30 минут**.
3. В фильтре найдите и выберите **нужную группу по названию**.
4. В результате отобразится актуальный **список процессоров**, входящих в эту группу.
5. **Выделите и скопируйте** весь отображённый список.

---

## 4. Фильтрация по ID процессоров

1. Вставьте скопированный список в поле **«ID процессоров»** на дашборде.
2. Дашборд отобразит **метрики только по процессорам выбранной группы**.

---

## Итого: краткий порядок действий

```
NiFi: Controller Settings → Reporting Task → указать группы
                        ↓
            Логи (обновление каждые 30 мин)
                        ↓
      Grafana → 30 мин → выбрать группу → скопировать список
                        ↓
      Вставить ID → получить мониторинг по группе
```

> **Помните:** держите группы в пределах 100 процессоров, иначе данные будут неполными.
